HTML
HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.
CSS
CSS stands for Cascading Style Sheets. It is used to describe the presentation of a document written in HTML or XML.

JavaScript is a programming language that is commonly used to create interactive effects within web browsers.
